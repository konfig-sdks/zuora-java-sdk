Submit a custom object bulk job

Submits a bulk job request for a bulk operation on the specified custom object records. A succcessful call returns a newly created bulk job. The job ID can be used to poll the job status or upload the CSV file if it is a &#x60;create&#x60; or &#x60;update&#x60; job.  ### Limits This custom object bulk jobs have the following limits: * The concurrent bulk job limit per tenant is 5. Bulk jobs in &#x60;accepted&#x60;, &#x60;pending&#x60;, or &#x60;in_progress&#x60; status are counted towards the concurrent bulk job limit. * The bulk job execuation order is not guaranteed, which means the bulk job that you submit ealier may be executed later.   * Only the users that have the \&quot;Delete Custom Objects\&quot; permission can create a &#x60;delete&#x60; bulk job. Only the users that have the \&quot;Edit Custom Objects\&quot; permission can create a &#x60;create&#x60; or &#x60;update&#x60; bulk job. See [Platform Permissions](https://knowledgecenter.zuora.com/Billing/Tenant_Management/A_Administrator_Settings/User_Roles/h_Platform_Roles#Platform_Permissions) for more information.

Upload a file for a custom object bulk job

Uploads a &#x60;.csv&#x60; file for a custom object bulk job to create or update custom object records. The job must be in &#x60;pending&#x60; status. Only one file is allowed per job. Each file supports only one type of operation, with an &#x60;id&#x60; column required for update operations. The job will start once the upload is finished.  Only the users that have the \&quot;Edit Custom Objects\&quot; permission can upload files to custom object bulk jobs. See [Platform Permissions](https://knowledgecenter.zuora.com/Billing/Tenant_Management/A_Administrator_Settings/User_Roles/h_Platform_Roles#Platform_Permissions) for more information.
