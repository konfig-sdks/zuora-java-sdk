List hosted pages

Returns the Payment Pages configuration metadata, specifically, page ID, page version, payment method type.  The following are the version-specific and general REST requests for Payment Pages:  * The request for Payment Pages 2.0 configuration information: &#x60;GET &lt;BaseURL&gt;/hostedpages?version&#x3D;2&#x60; * The request for all versions of Payment Pages configuration information: &#x60;GET &lt;BaseURL&gt;/hostedpages&#x60;  ### Notes If you do not have the corresponding tenant setting enabled, such as specifying the request &#x60;version&#x60; parameter to 2 but disabliing Payment Pages 2.0, you will receive an error.
