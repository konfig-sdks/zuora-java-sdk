Create a payment run

Creates a payment run. You can create a payment run to be executed immediately after it is created, or a scheduled payment run to be executed in future.  To filter the payments to be collected for a payment run, you can use either of the following methods but not both:    - Use the &#x60;accountId&#x60;, &#x60;batch&#x60;, &#x60;billCycleDay&#x60;, &#x60;currency&#x60;, &#x60;paymentGatewayId&#x60;, and &#x60;billingRunId&#x60; fields to define the billing documents to be collected.   - Use the &#x60;data&#x60; field to specify the records of accounts and billing documents to be collected, in a more flexible manner. See [Create custom payment runs through Zuora REST API](https://knowledgecenter.zuora.com/Billing/Billing_and_Payments/K_Payment_Operations/CA_Payment_Runs/Create_custom_payment_runs_through_Zuora_REST_API) for more information.  If no filter criteria are specified in the request body, the payment run collects payments for all accounts.
